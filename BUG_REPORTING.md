# Bug Reporting Guide

Thank you for helping improve Corrupted Core TD!

## Information to Include

### Game Version

Example:

Corrupted Core TD v1.0

### Description

Describe what happened.

### Steps to Reproduce

1. What did you do?
2. What happened?
3. Can it be reproduced consistently?

### Expected Behavior

What should have happened?

### Actual Behavior

What actually happened?

### Screenshots or Videos

Attach any screenshots or recordings if available.

### Additional Notes

Any other information that may help identify the issue.

## Feature Requests

Suggestions are welcome.

Please describe:
- The requested feature
- Why it would improve the game
- Any examples from other maps or games
